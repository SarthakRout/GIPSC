## To test regex for tokens
- From project root directory, use 
```python
python3 tests/Milestone2/test_regex.py 
```
or 

```python
make test
```
## To test source code for any program

- Run
```python
python3 src/Milestone2/lexer.py PATH-TO-SOURCE-CODE
```
